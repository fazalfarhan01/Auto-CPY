How to use

On Windows

  1. If you have enabled launch on connect, make sure you restart your computer for background service to run.
  2. If launch on connect is enabled, it is a must to add an antivirus exclusion.

On Android

  1. Turn on USB debugging from developer options. Search google on how to enable it.
  2. Make sure you accept the Allow USB Debugging prompt
  3. Re-Connect your android device
